디자인 정책 v3 (변경된 것만)

  • • 아이콘 스타일: Lucide stroke-only (1.6px), color #cdd9e5 (--icon-stroke), hover 시 #58a6ff (primary).

  • • fallback 아이콘: 매니페스트가 iconKey 매핑 없으면 plug 아이콘.
  • • 나머지: v2 와 동일 (5×2 grid, 10셀 후 scroll, installing overlay, "+" 외부 마켓플레이스).
