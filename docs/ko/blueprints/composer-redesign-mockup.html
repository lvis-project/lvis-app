주요 변경 vs v5

  • inline-status chip 제거 — ScrollArea 의 ReasoningCard 가 status 담당 (중복 제거)
  • 권한 + 권한 큐 승인 → TOP ROW 이전 — 첨부와 페르소나 사이. TOP = "환경 컨트롤" / BOTTOM = "Turn 컨트롤" 의미 분리
  • BOTTOM ROW 좌측 순서: TokenRing → 가이드 → 단축키 → (즉시) → grow → 취소 → 전송
  • 메시지 큐 명칭 — "큐" → "메시지 큐" (권한 큐와 구분)
  • BOTTOM ROW 가 2 cluster (info+actions) 로 단순화 — 3 cluster 였던 v5 의 state cluster 가 권한/승인 → top row 로 이전 + status 제거 → 사라짐
