Agent Hub Plugin UI v3 — 4줄 → 3줄 압축 + 리스크 통합

v2.1 의 4줄 콘텐츠를 3줄로 압축했습니다. 마이워크는 Row 1 = LLM 분석 + 승인 요청 / Row 2~3 = 좌측 stack(주간업무 + 내 보드 동일 폭) + 우측 오늘 일정 세로 stretch 구조, 팀보드는 팀 업무 요약을 최상단(Row 1, 2컬럼 narrative) → 좌측 stack(통합 대시보드 + 팀 보드/list 동일 폭) + 우측 오늘 팀원 전체 일정(세로 stretch) 의 3줄 구조입니다. 리스크는 전체 KPI / 팀원 / 이슈·리스크 paragraph 모두에 일관 색상으로 노출됩니다.

v3 합의 필요 항목

  • 마이워크 우측 사이드 컬럼의 폭(0.55fr) 이 compact viewport 에서 충분한지 — 더 좁히고 그 폭 만큼 LLM 카드를 늘릴지.
  • 승인 요청이 0건일 때: 카드 자체를 숨길지, 빈 상태로 남길지("대기 중인 승인 없음").
  • 팀원 mini row 의 리스크 0건 표시: 중성 회색(현재) vs 아예 미노출.
  • 팀 통합 대시보드 페이지네이션을 ‹ › 버튼 대신 가로 swipe 인지 / dot indicator 인지.
  • 팀 업무 요약 paragraph 의 길이 정책: 최대 N줄 truncation 후 "더 보기"인지, 자유 길이인지.
  • 오늘 팀원 전체 일정 카드의 참석자 avatar 최대 노출 수(현재 4) + "+9명" overflow 처리.

Scope notes · 구현으로 오해하지 말 것

  • 이 파일은 정적 docs/design artifact 이며 runtime/source/plugin 파일을 변경하지 않는다.
  • 3-row 압축은 IA 합의 산출물이며 실제 grid template / breakpoint 는 컴포넌트 구현 시 별도 결정.
  • 팀원 mini row 의 카운트 데이터 모델은 추후 Agent Hub API contract 확정 후 결정 (per-member aggregate).
  • 플러그인 버전 bump 는 필요 없다. docs/design 전용 변경이다.
