Plugin Grid V3

This English design artifact defines plugin browsing density, install and update affordances, state labels, and empty states. The Korean archive is available at docs/ko mirror.

Screen Contract

  • The artifact must be understandable as the default review surface without opening the archive.
  • Workflow-critical states must be explicit: empty, loading, active, error, success, and permission-blocked where applicable.
  • Renderer changes should preserve accessibility labels, keyboard behavior, and test hooks unless the owning test is updated.

Implementation Anchors

  • src/ui/renderer/components/
  • src/ui/renderer/tabs/
  • src/ui/renderer/__tests__/

Update Checklist

  • Update this artifact when layout, state naming, or primary actions change.
  • Keep screenshots, HTML mockups, and component tests aligned.
  • Do not use decorative-only changes to mask missing workflow states.
