# Code of Conduct

This project adopts the **Contributor Covenant**, version 2.1 — a widely used
community-standard code of conduct for open source projects.

The full text is available at:

> https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating in this project (filing issues, submitting pull requests,
joining discussions), you agree to abide by its terms.

## Scope

The Code of Conduct applies within all project spaces — the GitHub
repository, issue tracker, pull request reviews, and any public
communication channel where you represent the project.

## Enforcement

Reports of unacceptable behavior in project spaces may be sent to the
project maintainer at `jo.dreame@gmail.com`. All reports are reviewed and
investigated promptly and fairly.

The maintainer reserves the right to remove comments, commits, code,
issues, and other contributions that do not align with this Code of
Conduct, and to ban temporarily or permanently any contributor whose
behavior is deemed inappropriate, threatening, or unwelcome.
